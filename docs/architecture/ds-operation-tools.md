- 토큰/컴포넌트 사용 현황을 빌드타임에 json으로 저장하고 대시보드 API로 보내주는 웹팩 플러그인을 작성
- vernacular 사용자 대시보드에서 해당 디자인시스템을 어떤 프로젝트에서 얼마나 사용하고 있는지 확인 가능

- 의미있는 정보들
  - 디자인시스템 버전 (파편화 확인)
  - 인라인 스타일로 덮어쓴 것들 (계약 위반 탐지)
  - 어떤 컴포넌트들이 자주 함께 쓰이는지
  - 어떤 prop이 많이 쓰이고 덜 쓰이는지

- 이 정보들을 알게되면 뭘 할 수 있게 될지
  - 수집된 데이터 기반의 동적인 역방향 피드백
    - 린트로 아래와 같은 메시지 출력 가능
    - "이 컴포넌트는 다음 달 1일에 삭제될 예정입니다. 현재 프로젝트의 80%가 NewButton으로 교체되었습니다. [Quick Fix: NewButton으로 교체하기]"
